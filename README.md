# Kirjutamisstiili analüüs ja modelleerimine Discordi sõnumite põhjal

See koodivaramu sisaldab Tartu Ülikooli informaatika õppekava bakalaureusetöö  
**„Kirjutamisstiili analüüs ja modelleerimine Discordi sõnumite põhjal“** praktilise osa raames loodud koodi.

Kaustas `bot-components/` on Discordi botiga seotud kood. Kaustas `emulation/` on failid, mis on seotud LSTM-mudeliga. Kaustas `scripts/` asuvad kõik muud koodifailid, mille abil andmete töötlust ja analüüsi tehti.
